OpenAid Wysiwyg Feature

Feature Dependencies:

Modules Used:
emptyparagraphkiller - http://drupal.org/project/emptyparagraphkiller
wysiwyg - http://drupal.org/project/wysiwyg

Description:
The OpenAid Wysiwyg Feature enables the Wysiwyg module and configures it for use with the CKEditor library. It also enables a number of common default settings for both the Filtered HTML and Full HTML options. The Empty Paragraph Killer filter module is included to address a known bug in the CKEditor implementation that can lead to an empty paragraph tag inserted at the beginning of content.
